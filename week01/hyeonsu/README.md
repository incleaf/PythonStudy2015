### 복소수도 지원하는 파이썬

(6+9j)

### if 사용하기
```python
>>> a = 3
>>> if a > 1:
...     print("a is greater than 1")
...
a is greater than 1
```
print문 앞의 `...`은 아직 문장이 끝나지 않았음을 알려주는 프롬프트
### for 사용하기
```python
>>> for a in [1, 2, 3]:
...     print(a)
...
1
2
3
```

### while 사용하기
```python
>>> i = 0
>>> while i < 3:
...     i=i+1
...     print(i)
...
1
2
3
```

### 함수 맛보기
```python
>>> def sum(a, b):
...     return a+b
...
>>> print(sum(3,4))
7
```
